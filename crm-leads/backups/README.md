# Локальные резервные копии CRM

Файлы в этой папке, кроме README, игнорируются Git.

Полный backup включает:

- `crm_demo.dump` — заявки FastAPI;
- `twenty.dump` — PostgreSQL Twenty;
- `twenty-local-storage/` — локальные файлы Twenty;
- `metabase-data/` — настройки и дашборды Metabase.

Backup может содержать персональные данные и не должен публиковаться.

Отдельно в защищённом хранилище необходимо сохранить локальные `.env`, особенно `ENCRYPTION_KEY` Twenty. Потеря этого ключа сделает зашифрованные секреты Twenty недоступными после восстановления.
