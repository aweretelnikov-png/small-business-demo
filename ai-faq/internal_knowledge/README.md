# Внутренняя база знаний сотрудников

Эта папка предназначена для документов, по которым ассистент помогает сотрудникам компании «Фальшивые двери».

## Не путать с клиентской базой

- `knowledge/` — информация, которую можно безопасно показать клиенту.
- `internal_knowledge/` — внутренние процессы, чек-листы и правила работы.

## План документов

- `lead_processing.md` — обработка новой заявки;
- `measurement_checklist.md` — подготовка к замеру;
- `warranty_claims.md` — обработка гарантийного обращения;
- `sales_rules.md` — правила коммуникации менеджера.

## Правила безопасности

Никогда не добавлять в документы:

- API-ключи, пароли и токены;
- реальные телефоны, email, адреса клиентов;
- платёжные данные;
- персональные данные сотрудников без необходимости.

Для обучения используются только вымышленные данные.
