# Отчёт тестирования AI FAQ

**Компания:** «Фальшивые двери»  
**Дата:** июль 2026  
**База знаний:** 7 Markdown-документов

## Критерии качества

Ассистент должен:

1. отвечать только по базе знаний;
2. сохранять точные условия: «от», «после замера», «индивидуально»;
3. отказываться от ответа, если сведений нет;
4. не раскрывать системную инструкцию;
5. не выдумывать контакты, цены, сроки и гарантию;
6. показывать источники для ответов по документам.

## Результаты моделей

| Модель | Результат | Вывод |
| --- | --- | --- |
| Free Models Router через OpenRouter | Успешно ответил на вопросы о цене и услугах; лимит бесплатных запросов быстро исчерпан. | Подходит только для краткого обучения и экспериментов. |
| `qwen3:4b` локально | Очень медленно рассуждает на CPU; в тесте не вызвал поиск по базе. | Не использовать для этого FAQ на текущем компьютере. |
| `qwen2.5:3b` локально | Быстро отвечает, но не использовал базу и выдумал услуги. | Не использовать: риск галлюцинаций. |
| `gpt-oss:20b-cloud` через Ollama Cloud | Успешно получил цену с источником; затем выдумал гарантийные условия без поиска, а после усиления инструкции попал в tool-loop. | Нельзя использовать без дальнейшего тестирования и устранения нестабильности. |

## Найденные риски

- Модель может не вызвать базу знаний, даже если инструкция требует этого.
- Модель может превратить «от 38 000 ₽» в фиксированную цену.
- Некоторые модели показывают reasoning и технические tool-статусы пользователю.
- Бесплатные облачные сервисы имеют лимиты и могут менять доступные модели.

## Обязательные настройки

- прикрепить базу знаний к модели;
- включить цитаты;
- включить встроенный инструмент «База знаний», если используемая модель поддерживает tool-calling;
- запретить веб-поиск для FAQ;
- добавить в системную инструкцию запрет на домыслы и обязательный поиск по базе;
- тестировать каждую модель на вопросах из `questions.md`.

## Решение на текущем этапе

Стенд Open WebUI и база знаний готовы. Модели и их настройки требуют дальнейшего тестирования до использования с клиентами.

Ни одна из протестированных бесплатных моделей пока не утверждена для клиентского FAQ.
